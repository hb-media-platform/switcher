# Switcher
Switcher Form Builder Element

## NPM scripts

Development mode `npm run watch` or simply `npm start`
Development mode for IE `npm run watch:legacy`
Production build `npm run build`

## Installation

```html
    <switcher>
        <slot>Dunkles Thema</slot>
    </switcher>
```
